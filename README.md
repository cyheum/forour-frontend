# Project overview
#### 서비스 이름

#### 서비스 소개
* ##### 꽃 추천 서비스
  -. 선택한 기념일에 상대에게 선물하면 좋을 꽃을 추천해준다.  
  -. 주어진 상황에 상대가 했을 법한 행동을 선택한다. 선택한 내용을 바탕으로 상대의 성향을 분석하고, 어울리는 꽃을 추천해준다.
